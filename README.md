# BrightLearn-Data-Agent

Retail Sales Data Agent
Project Objective
The objective of this project was to build a working Retail Sales Data Agent on Databricks using a retail sales dataset.
The agent was designed to help business users analyse retail performance, answer business-related questions, and generate useful insights from sales data in simple business language.
This project focused on:
Understanding retail sales data
Creating a business-focused AI data agent
Writing custom agent instructions
Testing the agent with real business questions
Generating insights from retail transactions
Tools Used
Databricks
SQL
CSV Dataset
GitHub
Databricks Genie / AI Agent Workspace
Dataset Overview
The dataset used in this project was retail_sales_data.
The dataset contains retail transaction information with the following columns:
Transaction ID
Date
Customer ID
Gender
Age
Product Category
Quantity
Price per Unit
Total Amount
The dataset consisted of 1,000 retail transactions covering categories such as:
Electronics
Clothing
Beauty
The data was used to analyse:
Revenue trends
Customer demographics
Product performance
Sales behaviour
Monthly sales trends
Steps Followed
1. Upload Dataset
The retail sales CSV dataset was uploaded into Databricks.
2. Review Columns
The dataset columns were reviewed to understand the available business information.
3. Prepare Table
The data was prepared and organised into a usable table format for analysis.
4. Create Data Agent
A retail sales data agent was created in Databricks.
5. Write Agent Instructions
Custom instructions were written to guide how the agent should answer questions and analyse data.
6. Test with 10 Questions
The agent was tested using ten business-related retail questions.
7. Validate Responses
The generated answers were reviewed and validated against the dataset.
8. Create Documentation
A full write-up and documentation of the project was completed.
9. Push to GitHub
The completed project files were uploaded to GitHub.
10. Submit Repository Link
The GitHub repository link was submitted through the BrightLearn portal.
My Agent Instructions
The agent was instructed to:
Analyse retail sales information
Explain trends in simple business language
Help business users understand store performance
Focus on actionable insights
Avoid technical jargon
Avoid making assumptions about missing data
Only provide conclusions supported by the dataset
Ask for clarification if questions are unclear
Keep responses concise and professional
The dataset used by the agent was explicitly defined as retail_sales_data.

Key Insights
The dataset analysis produced the following insights:
The dataset contains a total of 1,000 transactions.
Overall sales revenue was approximately $456,000.
The average transaction value was approximately $456.
Electronics generated the highest revenue at approximately $156,905.
Clothing generated approximately $155,580 in revenue.
Beauty generated approximately $143,515 in revenue.
Customer distribution was balanced between genders:
510 Female customers
490 Male customers
The largest customer age group was 45–54 years.
Monthly sales peaked in May 2023 with approximately $53,150 in revenue.
Electronics showed strong overall business performance.
These insights helped identify customer behaviour patterns, top-performing categories, and overall sales performance.
