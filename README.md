# Retail Sales Data Agent

---

## Project Objective

The objective of this project was to build a functional Retail Sales Data Agent using Databricks and a retail sales dataset. The agent was designed to help business users analyse retail performance, answer business-related questions, and generate meaningful insights from sales data using simple business language.

This project focused on:

* Understanding retail sales data
* Building a business-focused AI data agent
* Writing custom AI agent instructions
* Testing the agent with real business questions
* Generating insights from retail transaction data

---

## Tools & Technologies Used

* Databricks
* SQL
* CSV Dataset
* GitHub
* Databricks Genie / AI Agent Workspace

---

## Dataset Overview

The dataset used in this project was `retail_sales_data`.

The dataset contained retail transaction information with the following columns:

* Transaction ID
* Date
* Customer ID
* Gender
* Age
* Product Category
* Quantity
* Price per Unit
* Total Amount

The dataset consisted of 1,000 retail transactions across categories such as:

* Electronics
* Clothing
* Beauty

The data was analysed to understand:

* Revenue trends
* Customer demographics
* Product performance
* Sales behaviour
* Monthly sales trends

---

## Project Workflow

### 1. Upload Dataset

The retail sales CSV dataset was uploaded into Databricks.

### 2. Review Dataset Structure

The dataset columns and data types were reviewed to understand the available business information.

### 3. Prepare the Dataset

The data was cleaned, organised, and prepared into a usable table structure for business analysis.

### 4. Create Retail Data Agent

A retail-focused AI data agent was created inside Databricks.

### 5. Write Agent Instructions

Custom instructions were written to guide how the agent should analyse data and respond to business-related questions.

### 6. Test the Agent

The agent was tested using multiple business-focused analytical questions.

### 7. Validate Responses

The generated answers were reviewed and validated against the dataset results.

### 8. Create Documentation

Complete documentation and project explanations were prepared.

### 9. Push to GitHub

The completed project files and documentation were uploaded to GitHub.

## Agent Instructions

The AI agent was instructed to:

* Analyse retail sales information
* Explain trends using simple business language
* Help business users understand retail performance
* Focus on actionable insights
* Avoid technical jargon
* Avoid assumptions about missing data
* Only provide conclusions supported by the dataset
* Ask for clarification if questions are unclear
* Keep responses concise and professional

The dataset used by the agent was explicitly defined as:

```sql
retail_sales_data
```

## Sample Business Questions Tested

The agent was tested using questions such as:

* What is the best-selling product category?
* What is the best-selling category by revenue?
* Which age generates the most revenue?
* Between ages 20 and 35, which age generates the highest sales?
* Which product category is most purchased by customers aged 20–25?
* What is the average transaction value?
* Which gender generates more sales?
* Show sales trends over time.
* Which product category performs the worst?
* Which product category had the highest and lowest revenue?

## Key Insights Generated

The analysis produced several business insights:

* The dataset contained a total of 1,000 transactions
* Overall sales revenue was approximately **$456,000**
* Average transaction value was approximately **$456**
* Electronics generated the highest revenue at approximately **$156,905**
* Clothing generated approximately **$155,580** in revenue
* Beauty generated approximately **$143,515** in revenue

#### Customer Distribution

* 510 Female customers
* 490 Male customers

#### Customer Behaviour Insights

* The largest customer age group was **45–54 years**
* Monthly sales peaked in **May 2023** with approximately **$53,150** in revenue
* Electronics showed the strongest overall business performance

These insights helped identify:

* Customer purchasing behaviour
* Top-performing categories
* Revenue trends
* Overall business performance
